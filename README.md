# TEAM : CS Shopping Mall
---
Java로 제작한 온라인 쇼핑몰 사이트 입니다.

## 주요 기능 요약
- 현재 인기 랭킹 출력

- 카테고리 별 검색 기능

- 실시간으로 바뀌는 배송조회 시스템
